== Description ==

Send email alerts to developer when object caching is turned off on the site. Require functioning of WP cron + wp_mail()

Enter emails as an option with key "object_cache_off_alert_emails"

Please report any issues directly to Github repo - https://github.com/ashfame/object-cache-off-alert/

Pull requests are welcome :-)

== Installation ==

* Upload the folder `email-cop` to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
